# psxemu
attempt at a psx emulator to better understand lower level stuff smh

# Currently Implemented
- Entire CPU
- RAM Reads/Writes
- GPU Commands

# TODO

- [SDL2](https://www.libsdl.org/)
- [sdl-gpu](https://github.com/grimfang4/sdl-gpu)
